# AGENTS.md 是项目的统一上下文索引文件
以下是开发过程中可能要用到的上下文的入口

## .agents/local-docs-index.md
- 描述：这是一个索引文件，用于索引项目内的一些总结/复盘/归档类的文档**目录**，用于给开发功能时提供参考
- 用法：读取这个索引文件，找到其中提到的目录，然后查看这些目录中包含的文档，然后再读取具体的文档内容

## .agents/remote-docs-index.md
- 描述：这是一个索引文件，用于索引远程的文档 URL 以及如何获取远程文档中的内容，用于给开发功能时提供参考
- 用法：读取这个索引文件，找到其中的远端文档的 URL，根据其说明的获取方式（直接通过 URL 获取内容/通过 CLI 等特定方式），读取具体的文档内容

## .agents/constraints.md
- 描述：这是一个具体的文档，项目的开发约束都会写在这个文档内，用于规范开发
- 用法：直接读取这个文件中的内容，严格遵守其中的规范和约束
